DLL Tracker API

Welcome to the DLL Tracker API. Use the endpoints below to access team and match data.

Available Endpoints

GET /team-info

Get comprehensive team statistics, match history, and form.

Query parameter: team_id (required)

POST /create-match

Create a new match with team and player information.

GET /match-result

Get match results using the match code.

Query parameter: match_code (required)

GET /match-stats

Get detailed match statistics using the match code.

Query parameter: match_code (required)

GET /status

Check the API status.
